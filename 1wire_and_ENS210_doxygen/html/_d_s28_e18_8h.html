Enumerator
READ_ROM 

can only be used if there is a single slave on the bus

MATCH_ROM 

address a specific slave by ROM ID

SEARCH_ROM 

enumeration of all slaves on bus

SKIP_ROM 

don't use ROM ID (applicable only when there is only 1 slave on the bus)

OVERDRIVE_MATCH 

selected (matched ROM ID) slave goes into overdrive

◆ DS28E18_utility_delay_T

enum DS28E18_utility_delay_T

DS28E18 delay command argument is an exponent: Delay is 2^arg in msec. The actual delay time is from 1ms to 32s respectively Code relies on enum defined as the exponent as below.
